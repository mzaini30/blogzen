Membuat server SPA dengan Javascript

Sekarang dengan Javascript

Setelah sebelumnya aku menjelaskan tentang cara membuat server SPA dengan Go, sekarang kita buat dengan Javascript.

Oh iya, kode di sini aku copas dari Google. Yang penting sih kita paham aja cara kerjanya.

Pertama, kita install dulu Express dengan PNPM:

pnpm add express

Baru setelah itu, kita buat file server.js yang isinya:

/*
  Incredibly simple Node.js and Express application server for serving static assets.

  Given as an example from the React Router documentation (along with examples
  using nginx and Apache):
  - https://github.com/ReactTraining/react-router/blob/master/docs/guides/Histories.md#browserhistory
*/

const express = require('express');
const path = require('path');
const port = process.env.PORT || 2020
const app = express();

// serve static assets normally
app.use(express.static(__dirname + '/'));

// handle every other route with index.html, which will contain
// a script tag to your application's JavaScript file(s).
app.get('*', function (request, response) {
  response.sendFile(path.resolve(__dirname, 'index.html'));
});

app.listen(port);
console.log("server started on port " + port);

Untuk menjalankannya, cukup dengan perintah:

node server.js

Nanti, situsnya bisa dibuka di localhost:2020.
