Membuat domain gratis JS.ORG

Kalau bisa gratis, kenapa harus bayar?

Domain itu seperti .com, .net, dan .js.org. Cuma, yang membedakan JS.ORG dengan dotcom dan dotnet adalah karena dia gratis. Tentu, nggak akan dilewatkan dong sesuatu yang namanya gratis itu.

Kenapa bisa gratis? Karena domain ini didukung oleh Cloudflare buat semua komunitas JavaScript. Makanya, nama domainnya ini JS.ORG. JS, singkatan dari JavaScript. Tapi, nggak mesti komunitas JavaScript sih. Blog biasa juga bisa. Yang penting ya hostingnya di Github atau Gitlab.

Dan sekarang, kita mulai bahas aja yang kita tunggu-tunggu tadi, yaitu cara mendaftar domain JS.ORG:

  1. Pastikan websitemu dihosting di Github
    atau Gitlab
  2. Tentukan nama domainnya. Misalnya aja mau halo.js.org, maka buat repostiori halo di Githubku
  3. Buat file bernama CNAME yang isinya halo.js.org
  4. Terus daftarkan di JS.ORG dengan format seperti ini: "halo": "namamu.github.io/halo",

Mudah aja kan cara daftarnya? Ayo deh langsung aja dicoba.
