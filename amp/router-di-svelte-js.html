Router di Svelte JS

Mencoba menerapkan keunggulan SPA!

Router adalah hal yang sangat esensial bagi suatu website yang menerapkan single page app (SPA). Karena dengan router, kita bisa memanipulasi URL di website SPA yang kemudian akan kita jalankan fungsi-fungsi tertentu sesuai dengan rules URL yang telah kita atur. Contoh dari SPA adalah halaman:

  • situs.com
  • situs.com/about
  • situs.com/anggota/200

Semua halaman tersebut sebenarnya hanyalah satu halaman yang sama namun menampilkan halaman yang saling berbeda dikarenakan sudah diatur oleh routernya.

Ada dua macam router yaitu yang menggunakan hash sama yang nggak. Kalau menggunakan hash (contoh: situs.com/#/daftar), cocok pakai svelte-spa-router. Sedangkan kalau nggak pakai hash, pakai page.

Menggunakan Svelte SPA Router

Instalasi

npm i svelte-spa-router

Import halaman-halaman untuk router

Ini adalah contoh ketika dipanggil di dalam App.svelte:

<Router {routes}/>

<script type="text/javascript">
    import Router from 'svelte-spa-router'

    import Beranda from './halaman/Beranda.svelte'
    import Tambah from './halaman/Tambah.svelte'
    import Edit from './halaman/Edit.svelte'

    const routes = {
        '/': Beranda,
        '/tambah': Tambah,
        '/:id': Edit
    }
</script>

Contoh penggunaan params

<p>Hello {params.nama}</p>

<script type="text/javascript">
    export let params
</script>

Push router

<script type="text/javascript">
    import {push} from 'svelte-spa-router'
    const pindah = () => push('/about-me')
</script>

<div on:click={pindah}>Klik ini</div>

Push router dengan <a>

<a href='/#hello'>Hello</a>

Menggunakan Page

Instalasi

npm i page

Kemudian, buka package.json dan cari tulisan:

"start": "sirv public"

Nah, itu diganti dengan:

"start": "sirv public --single"

Import halaman-halaman untuk router

Ini adalah contoh ketika dipanggil di dalam App.svelte:

<script>
 import router from "page"

 import Beranda from "./halaman/Beranda.svelte"
 import Baca from "./halaman/Baca.svelte"
 import Tulis from "./halaman/Tulis.svelte"
 import NotFound from "./halaman/NotFound.svelte"

 let page
 let params

 router("/", () => page = Beranda)
 router("/tulis", () => page = Tulis)
 router("/:slug", (x, next) => {
     params = x.params
     next()
 }, () => page = Baca)
 router("/*", () => page = NotFound)

 $: if(params){
  window.scrollTo(0, 0)
 }
</script>

<svelte:component this={page} {params}/>

Contoh penggunaan params

<script type="text/javascript">
    export let params
</script>

<p>Hello {params.nama}</p>

Push router

<script type="text/javascript">
    import router from "page"
    const pindah = () => router.replace('/about-me')
</script>

<div on:click={pindah}>Klik ini</div>

Push router dengan <a>

<a href='/hello'>Hello</a>
