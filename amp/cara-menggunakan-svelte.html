Cara menggunakan Svelte

Membuat website lebih cepat, ringan, dan menyenangkan

Instalasi

npx degit sveltejs/template nama-proyekmu

Kemudian, akan terdownlod tuh template Svelte di laptopmu. Nah, sekarang masuk ke nama-proyekmu dan jalankan:

npm i

Setelah selesai instalasi, jalankan:

npm run dev

Nah, sekarang kamu buka localhost:5000. Lalu, akan muncul Hello World ala Svelte...
