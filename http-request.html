HTTP Request

HTTP Request berarti melakukan request (permintaan) ke sebuah source melalui jalur HTTP. Jadi, nggak melalui jalur lain seperti melalui socket, SSH, dan jalur lainnya.

Untuk melakukan HTTP Request, kita biasanya menggunakan Fetch API seperti contoh di bawah ini:

let data = await fetch("https://source.com")
data = await data.json()
console.log(data)
