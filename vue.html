Vue

Vue merupakan framework JavaScript yang dibuat oleh Evan Yu yang sintaksnya itu sama seperti HTML. Contoh dari Vue adalah seperti ini:

<script setup>
import { ref } from "vue";

const nama = ref("Zen");
</script>

<p>Halo {{nama}}</p>

<style scoped>
p {
  background: blue;
}
</style>
